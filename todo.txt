Előkészületek:
==============
1. Backend vizsga starter repo klónozása, átnevezése: git clone starter_repo_URL
2. Node.js csomagok telepítése: npm i
3. Források másolása
4. Mongo szerver indítása: cmd -> startMongoDB.bat futtatása
5. VS CODE indítása, frissítések, ajánlott csomagok telepítése
6. Csoportos átnevezések:
         AdatbázisNeve, TáblaNeve1, TáblaNeveN, "/api/xyz"
7. Forrás JSON állomány átnevezése: "db_n.json" -ra

Feladatmegoldás:
================
1. "1" oldali tábla modelljének aktualizálása: oneside.model.ts
2. "N" oldali tábla modelljének aktualizálása: nside.model.ts
3. "1" oldali tábla elkészítése: "db_1.json"
4. Adatbázis, táblák létrehozása, mintaadatok feltöltése: .\db.bat
5. Fordítás indítása ("npm run dev" vagy Ctrl-Shift-B)
6. Thunder Client teszt kollekció importálása, aktualizálása
7. "N" oldali kontroller (route-ok, handlers) aktualizálása - Thunder Client tesztek
